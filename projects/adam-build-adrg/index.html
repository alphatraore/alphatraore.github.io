Mini ADSL R program (SDTM → ADaM + traceability)

Built ADaM datasets from SDTM with clear traceability and documentation.

What it does

  • Builds ADAM domains (ADSL, ADSV, ADAE)
  • Implements standard derivations (pop flags, baselines, visits)
  • Maintains traceability (source → derivation → analysis)
  • Produces ADRG-style documentation and metadata notes

Tools: SAS / R, CDISC ADaM
